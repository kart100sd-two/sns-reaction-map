# 記事作成手順: 高市総理・文春中傷動画問題

作成日: 2026-06-13

## 目的

高市総理・文春中傷動画問題を最初のサンプル記事として使い、今後すべての論点整理記事に使える標準ルートを作る。

この記事の役割は、誰かを断罪することではなく、読者が次を理解できるようにすること。

- 何が報じられたのか
- 何が確認済みで、何が未確認なのか
- どこが争点なのか
- 擁護側と批判側は何を根拠にしているのか
- 玉木代表のような関連事例と、どこが同じでどこが違うのか

## 完成記事の基本構成

```text
タイトル
リード文
30秒要約
確認済み事実
未確認・要検証ポイント
争点マップ
擁護側の意見
批判側の意見
関連事例との比較
時系列
根拠リンク
編集部メモ
```

## Step 1: 入力URLを集める

### 必須URL

| 種別 | URL | 用途 |
| --- | --- | --- |
| 主報道 | https://bunshun.jp/articles/-/89485 | 高市氏側の文春報道の要旨 |
| 関連報道 | https://times.abema.tv/articles/-/10251745 | 玉木氏側の説明記事 |
| 本人発信 | https://x.com/tamakiyuichiro/status/2064581187168027048 | 玉木氏本人の説明 |

### 追加で探すURL

| 種別 | 探すもの |
| --- | --- |
| 高市氏側の反論 | 公式会見、国会答弁、事務所コメント、X投稿 |
| 文春続報 | 第1弾〜第6弾の記事、証拠メール、音声、動画関連の記事 |
| 他媒体報道 | NHK、朝日、毎日、読売、産経、日経、ABEMA、TBS、FNN等 |

## Step 2: ソース台帳を作る

各URLを次の形式で記録する。

| 項目 | 記入例 |
| --- | --- |
| source_id | source-001 |
| title | 高市陣営「中傷動画」とは？ |
| url | https://bunshun.jp/articles/-/89485 |
| publisher | 文春オンライン |
| published_at | 2026-06-12 |
| source_type | 報道 |
| evidence_level | medium |
| summary | 文春が報じた中傷動画問題のまとめ記事 |
| notes | 文春側の主張。反論や追加報道の確認が必要 |

### evidence_levelの目安

| レベル | 意味 |
| --- | --- |
| high | 公式資料、国会会議録、本人発言、一次資料 |
| medium | 報道、調査記事、複数証言を伴う記事 |
| low | SNS投稿、匿名証言、切り抜き |
| unknown | 出典不明、スクショのみ、又聞き |

## Step 3: 事実・主張・推測を分離する

AIに最初にやらせる処理は、要約ではなく分類。

### 分類ルール

| ラベル | 意味 |
| --- | --- |
| 確認済み事実 | URL内で明確に確認できる出来事、発言、投稿 |
| 報道上の主張 | 報道機関が取材に基づき主張している内容 |
| 本人説明 | 当事者本人が説明している内容 |
| 本人否定 | 当事者本人が否定している内容 |
| 未確認 | 根拠不足、第三者確認不足、SNS上の推測 |
| 編集部分析 | 上記をもとにした編集側の整理 |

### この事例での分類例

| ラベル | 内容 |
| --- | --- |
| 報道上の主張 | 文春は、高市陣営が中傷動画を大量作成・拡散したと報じている |
| 報道上の主張 | 文春は、公設第一秘書が依頼し、松井健氏が作成・拡散を担ったと報じている |
| 本人説明 | 玉木氏は、松井氏と面識があり、紹介会社に12本委託したと説明 |
| 本人否定 | 玉木氏は、中傷動画依頼、有償拡散、ネット戦略関与を否定 |
| 未確認 | 高市氏本人の直接指示・認識 |
| 未確認 | 玉木氏側の12本の具体的内容、契約、費用 |

## Step 4: 争点マップを作る

争点は「賛成/反対」ではなく、問いの形にする。

| 争点 | 説明 |
| --- | --- |
| 陣営主導だったのか | 公設秘書の依頼が事実なら、支援者の自主活動とは言いにくい |
| 高市氏本人の認識はあったのか | 本人の直接指示・把握の有無は責任の重さに影響する |
| 政治批判と中傷の境界はどこか | 政策批判は許容されるが、人格攻撃や虚偽なら問題性が高い |
| 有償拡散・ステルス広報だったのか | 発信主体や対価が見えない場合、有権者が誤認し得る |
| 玉木氏側も同じ問題なのか | 同じ松井氏が出てくるが、関与内容と根拠の強さは異なる |

## Step 5: 賛否意見を作る

このサービスでは「賛成」「反対」よりも、「擁護」「批判」「保留」に分ける。

### 擁護側

| 主張 | 根拠 | 弱点 |
| --- | --- | --- |
| 高市氏本人の直接指示は未確認 | 文春報道でも本人指示の確認とは別問題 | 公設秘書の関与が事実なら管理責任は残る |
| 政治批判動画と中傷動画の境界は慎重に見るべき | 選挙では政策・能力批判も許容される | 大量匿名拡散や虚偽なら問題性が高い |
| 玉木氏側を高市氏側と同一視するのは雑 | 玉木氏は通常委託を認め、中傷依頼を否定 | 委託内容の第三者確認は必要 |

### 批判側

| 主張 | 根拠 | 弱点 |
| --- | --- | --- |
| 公設秘書の依頼が事実なら陣営問題として重大 | 文春が公設第一秘書の依頼を報じている | 高市氏本人の認識は別途確認が必要 |
| 大量の中傷動画拡散は民主的な議論を歪める | 文春は1日100〜200本程度と報じている | 投稿実数や運用主体の確認が必要 |
| 玉木氏側も透明性の説明は必要 | 松井氏との面識と紹介会社への委託は本人が認めている | 中傷依頼と同列に扱う根拠は弱い |

### 保留・中立

| 主張 | 理由 |
| --- | --- |
| 現時点では文春報道と当事者説明の照合が必要 | 高市氏側の反論、国会答弁、契約・対価情報が必要 |
| 玉木氏側は貰い事故に近いが、説明責任は残る | 接点はあるが、中傷動画関与の根拠は確認できていない |

## Step 6: 関連事例比較を入れる

この事例では、玉木代表の件を関連比較として扱う。

| 比較軸 | 高市総理側 | 玉木代表側 |
| --- | --- | --- |
| 問題の性質 | 文春が報じた中傷動画作戦疑惑 | 松井氏との接点から波及した説明責任問題 |
| 松井氏の役割 | 文春は作成・拡散担当と報道 | 紹介者。紹介先の制作会社へ12本委託 |
| 中傷動画との関係 | 文春が中傷動画作成・拡散を報道 | 玉木氏は中傷動画依頼を否定 |
| 根拠の強さ | 文春が複数資料・証言を報道 | 本人説明中心。中傷関与の根拠は未確認 |
| 暫定評価 | 重大疑惑 | 貰い事故に近いが透明性説明は必要 |

## Step 7: タイトル案を作る

煽りすぎず、読者が論点を理解できるタイトルにする。

### 良いタイトル

- 高市陣営「中傷動画」問題とは何か 松井健氏をめぐる論点整理
- 高市氏と玉木氏は「同じ問題」なのか 松井健氏関連報道を比較する
- 中傷動画、外部委託、説明責任 高市氏・玉木氏の違いを整理する

### 避けるタイトル

- 高市も玉木も同罪か
- ネット工作の黒幕は誰だ
- 松井健氏が操った政治家たち

理由: 未確認情報の断定、名誉毀損、煽り表現になりやすい。

## Step 8: 公開前レビュー

### チェックリスト

- 文春の主張を事実として断定していないか
- 高市氏本人の直接指示を未確認として扱っているか
- 玉木氏側を同列に断定していないか
- 本人否定を掲載しているか
- SNS上の憶測と確認済み情報を分けているか
- 個人攻撃表現を引用しすぎていないか
- 根拠URLが付いているか
- 訂正可能な構造になっているか
