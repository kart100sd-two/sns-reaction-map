# 高市陣営「中傷動画」問題とは何か 松井健氏をめぐる論点整理

ステータス: 調査中  
主要ラベル: 報道あり / 未確認あり / 関連事例あり / SNS推論あり / 解説動画あり

## 30秒要約

週刊文春は2026年6月12日、高市陣営が自民党総裁選や衆院選で、ライバル候補や野党を中傷するショート動画を大量に作成・拡散していたと報じた。記事では、高市首相の公設第一秘書が動画作戦を依頼し、AIなどに詳しい起業家の松井健氏が作成・拡散を担ったとされている。

この問題の焦点は、単なる政治広報や政策批判なのか、陣営主導の中傷・ステルス的なネット工作なのかという点にある。高市氏本人の認識や直接指示、対価、運用アカウント、選挙法上の評価は追加確認が必要である。

また、松井氏と面識があり、紹介された動画制作会社に「たまきチャンネル」12本を委託したと説明している国民民主党の玉木雄一郎代表も、SNS上で関連事例として言及されている。ただし、玉木氏は中傷動画の制作・発信依頼、有償拡散、国民民主党のネット戦略への松井氏関与を否定しており、高市氏側の文春報道と同列に断定する根拠は現時点では弱い。
