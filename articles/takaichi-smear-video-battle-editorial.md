# 私ならこう見る 高市陣営「中傷動画」問題

最終更新: 2026-06-14  
ステータス: 調査中  
形式: 論点バトル / 編集者視点

## まず、どこから見るか

私なら、この件は「高市氏が悪いのか」からは入りません。

まず見るのは、誰が動画を作らせたのか。次に、誰の資金や指示で動いたのか。そして最後に、有権者から見て、その発信主体が分かる状態だったのか。

この順番で見ると、文春が報じた「公設秘書の関与」はかなり重要です。一方で、高市氏本人の直接指示を確認しないまま、本人が全部やったように語るのも危うい。

玉木氏の件も同じです。松井健氏との接点はある。だから説明は必要。でも、中傷動画依頼まで確認できていないなら、高市氏側と同じだと断定するのは早い。

SNSでは、複雑な話ほど一言で片づけられます。

- 高市氏はアウト
- 文春の捏造
- 玉木氏も同じ
- いや玉木氏は貰い事故

どれも分かりやすい。でも、分かりやすい話ほど、だいたい何かを落としています。

なので、この記事では感情ではなく、論点ごとに根拠の強さを見ます。

## この問題を3つに分ける

- 文春は、高市陣営の中傷動画作成・拡散疑惑を報じた。
- 報道では、公設秘書の依頼と松井健氏の関与が焦点になっている。
- 玉木氏にも松井氏との接点から話が飛び火したが、中傷動画依頼は否定している。

ここを混ぜると、話が一気に雑になります。

高市氏側は重大疑惑。  
玉木氏側は巻き込まれ型の説明責任問題。  
そして全体の本丸は、政治動画の外注とネット世論形成の透明性です。

# Round 1: 公設秘書が関わっていたなら、本人の責任は？

## 批判側の見方

公設秘書が動画作戦を依頼したと報じられている以上、これは単なる支援者の自主活動ではない。本人が直接指示していなくても、陣営の中枢に近い人物が関わっていたなら、政治責任や説明責任は重い。

## 擁護側の見方

高市氏本人の直接指示や認識は、まだ確認されていない。秘書や外部協力者の行為を、ただちに本人の指示とみなすのは早い。誰が、いつ、何を、どこまで依頼したのかを確認すべきだ。

## 私ならこう整理する

公設秘書の関与が事実なら、これは軽くありません。少なくとも「知らなかった」で終わる話ではない。

ただし、本人の直接指示と、陣営管理責任は分けた方がいいです。ここを混ぜると、批判も雑になります。

| 観点 | 整理 |
| --- | --- |
| 公設秘書の関与 | 事実なら重い |
| 高市氏本人の直接指示 | 追加確認が必要 |
| 説明責任 | 残る |

根拠の強さ:

| 立場 | 強さ |
| --- | --- |
| 批判側 | ★★★★☆ |
| 擁護側 | ★★★☆☆ |

# Round 2: 文春報道は信用できるのか？

## 批判側の見方

文春は、動画、メール、LINE/Signal、Zoom音声、証言などを報じたとしている。これだけ材料が出ているなら、少なくとも調査と説明は必要だ。

## 擁護側の見方

文春や共同通信の報道には、時系列や証拠の扱いに疑義があるという反応も多い。報道を根拠に、本人の関与や違法性まで一気に断定するのは危険だ。

## 私ならこう整理する

文春報道は無視できません。ここまで具体的に報じているなら、説明を求めるのは自然です。

一方で、報道は報道です。裁判所の認定でも、公式調査の結論でもない。だから「文春が報じたから全部事実」とするのも違う。

ここは、文春の提示材料、当事者の反論、国会答弁、他媒体の検証を並べて見るべきです。

| 観点 | 整理 |
| --- | --- |
| 文春報道 | 具体性があり、無視できない |
| 報道への疑義 | 検証対象として扱うべき |
| 現時点の結論 | 断定ではなく、説明要求の段階 |

根拠の強さ:

| 立場 | 強さ |
| --- | --- |
| 批判側 | ★★★★☆ |
| 擁護側 | ★★★☆☆ |

# Round 3: 政治批判と中傷の境界

## 批判側の見方

ライバル候補や野党を攻撃する動画が大量に作られ、拡散されていたなら、民主的な議論を歪める。発信主体が見えない形で流れていたなら、有権者は誰のメッセージか判断できない。

## 擁護側の見方

選挙では、政策や候補者の資質を批判する表現も許容される。すべての批判動画を「中傷」と呼ぶと、政治的批判そのものを萎縮させる危険がある。

## 私ならこう整理する

ここは動画の中身を見ないと判断できません。

政策批判なら許容範囲です。候補者の発言や政策への批判も、選挙では当然あり得る。

でも、虚偽、人格攻撃、発信主体を隠した大量拡散、有償のステルス広報が絡むなら、話は変わります。

つまり問題は「動画を作ったこと」ではなく、「何を、誰の指示で、誰の金で、誰の発信として見せたのか」です。

| 観点 | 整理 |
| --- | --- |
| 政策批判 | 許容され得る |
| 虚偽・人格攻撃 | 問題性が高い |
| 匿名大量拡散 | 透明性の問題 |
| 有償ステルス広報 | さらに重い |

根拠の強さ:

| 立場 | 強さ |
| --- | --- |
| 批判側 | ★★★★☆ |
| 擁護側 | ★★★★☆ |

# Round 4: 玉木氏も同じ問題なのか？

## 批判側の見方

玉木氏も松井氏と面識があり、松井氏から紹介された動画制作会社に「たまきチャンネル」12本を委託したと説明している。高市氏側を批判するなら、玉木氏側も同じ基準で委託内容や費用を説明すべきだ。

## 擁護側の見方

玉木氏側は、松井氏との接点と動画制作委託は認めているが、中傷動画の制作・発信依頼、有償拡散、国民民主党のネット戦略への関与は否定している。通常の動画制作委託と中傷動画疑惑を同列に扱うのは雑だ。

## 私ならこう整理する

玉木氏側は、かなり「貰い事故」に近いと思います。

同じ松井氏の名前が出てくるので、SNSで飛び火するのは分かる。でも、高市氏側は文春が中傷動画の大量作成・拡散疑惑として報じている。一方、玉木氏側は本人が通常委託を認め、中傷動画依頼は否定している。

この2つを同じ箱に入れるのは簡単ですが、根拠の強さが違います。

ただし、玉木氏側も完全に無関係ではありません。面識と委託は認めているので、委託した12本の内容、費用、時期について説明すれば、かなり整理されるはずです。

| 観点 | 高市氏側 | 玉木氏側 |
| --- | --- | --- |
| 松井氏との関係 | 作成・拡散担当と報道 | 面識、紹介会社への委託 |
| 中傷動画との関係 | 文春が報道 | 本人が依頼を否定 |
| 根拠の強さ | 報道資料・証言あり | 本人説明中心 |
| 暫定整理 | 重大疑惑 | 巻き込まれ型の説明責任問題 |

| 立場 | 強さ |
| --- | --- |

# Round 5: 本丸はネット選挙の透明性

## 批判側の見方

政治家や陣営が、外部制作者や別アカウントを使って動画を拡散する場合、有権者から見ると誰の発信なのか分かりにくい。これはステルス広報や世論操作に近づく危険がある。

## 擁護側の見方

SNS時代に、外部の制作会社や支援者が政治動画を作ること自体は珍しくない。問題にすべきなのは、外注そのものではなく、動画内容、対価、発信主体の表示、選挙期間との関係だ。

## 私ならこう整理する

ここが一番大事だと思います。

政治家が動画を作ること自体は普通です。外注も普通です。支援者が応援動画を作ることもあります。

でも、外注、匿名アカウント、ショート動画、AI、政治広告、支援者投稿が混ざると、見る側には誰のメッセージか分からなくなる。

この不透明さが、今回の疑惑を大きくしています。

| 問い | なぜ重要か |
| --- | --- |
| 誰が企画したのか | 陣営主導か自主活動かが変わる |
| 誰が費用を払ったのか | 有償広報・選挙運動の評価に関わる |
| どのアカウントで流したのか | 発信主体が見えていたかに関わる |
| 動画内容は何か | 批判か中傷かを分ける |
| 選挙期間中か | 法的・政治的評価に関わる |

根拠の強さ:

| 立場 | 強さ |
| --- | --- |
| 透明性を問題視する見方 | ★★★★☆ |
| 外注自体は問題ないという見方 | ★★★☆☆ |

# 総合整理

## 批判側が強いところ

- 公設秘書の関与が事実なら、陣営問題として重大。
- 中傷動画の大量作成・拡散が事実なら、選挙の公正性や民主的議論に関わる。
- 発信主体や対価が見えない政治動画は、ステルス広報化する危険がある。

## 擁護側が強いところ

- 高市氏本人の直接指示や認識は未確認。
- 文春・共同の報道には検証や反論の余地がある。
- 玉木氏側を高市氏側と同列に扱う根拠は弱い。

## 私の暫定結論

高市氏側は、重大疑惑として扱うべきです。特に公設秘書の関与が事実なら、本人の直接指示が未確認でも、説明責任はかなり重い。

一方で、本人が直接指示したかどうか、動画の中身が本当に中傷と言えるか、対価や発信主体はどうなっていたかは、まだ分けて確認すべきです。

玉木氏側については、現時点では貰い事故に近い。ただし、松井氏との接点と動画制作委託は本人が認めているので、委託内容を明らかにすることで疑念を減らせると思います。

この件の本丸は、政治家個人の好き嫌いではありません。

政治動画が、誰の指示で、誰の資金で、誰の発信として流れているのか。  
そこが見えないまま選挙や世論形成に使われることです。
