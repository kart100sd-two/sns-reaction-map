# 投票質問・選択肢（Axes）設計ガイドライン ＆ 既存テーマ改善案

本ドキュメントは、SNS反応マッププロジェクトにおける「ユーザー向け質問文・投票導線」の品質向上を目的とした設計ガイドライン、および既存6テーマに対する具体的な改善案をまとめたものです。

---

## 1. 投票質問・選択肢（Axes）設計ガイドライン

多くの来訪者が直感的に選択でき、かつモバイル画面での視認性・操作性を担保するための4つのコアルールを策定しました。

### ① 当事者意識（フック）の醸成：「ジブンゴト」化ルール
ユーザーが「これは私の意見だ」「この立場なら表明したい」と直感的に思えるよう、主観的な価値観や一次感情に紐づけた表現を採用します。

*   **「〜すべき」から「〜を優先したい/〜を守りたい」へ**
    *   客観的・政策的な命令口調（上からの視点）ではなく、個人のスタンス・ポリシー（主体の視点）に主語を置き換えます。
    *   *例:* 「AI規制を強化すべき」 ➡ 「クリエイターの権利と生活を守りたい」
*   **一次感情（不安・期待・不満）を内包する**
    *   単なる論理的賛否ではなく、「〜が心配だ」「〜は理不尽だ」「〜を楽しみたい」といった、SNSユーザーが発露しやすい生の声に近い感情を含めます。

### ② 脱・システム用語：心理的スタンスへの昇華ルール
分析・開発側で定義される「その他」「ノイズ」「派生論点」「別論点接続」といった無機質な分類名をそのままUIに出さず、ユーザーが自己表明として選びやすい文脈へと昇華させます。

*   **「保留・その他」の再定義**
    *   単に「わからない」と切り捨てるのではなく、「なぜ決められないのか」「どういう態度なのか」を心理的に分析し、選択肢化します。
    *   *代替表現の例:*
        *   **情報・判断材料の不足:** 「まだ情報が足りず、どちらとも言えない」
        *   **対立への忌避感:** 「両者の対立が激しすぎて、少し様子を見たい」
        *   **当事者性の薄さ:** 「自分の生活から遠く、ピンとこない」
*   **「派生論点・別論点」の再定義**
    *   「別の論点が気になる」とするのではなく、ユーザーが「本質はここではない」と感じる具体的なポイントに光を当てます。
    *   *代替表現の例:* 「本質は〇〇の議論であり、今の争点には納得がいかない」

### ③ モバイルファースト：文字数＆レイアウト制限ルール
スマートフォンの縦画面で投票ボタンが並んだ際、不自然に折り返して崩れたり、文字で埋め尽くされて視認性が下がるのを防ぎます。

*   **文字数の目安:** **15文字〜25文字**（最大30文字厳守）
*   **表現の簡素化:**
    *   助詞（〜の、〜において）を極力削り、体言止めや短い動詞で結びます。
    *   読点（、）は1文の中に原則1つまでとし、可能な限り排除して一息で読めるコピーにします。

### ④ 背景説明（Intro）のコピライティング：中立と感情のフック
投票の前提となるトピック説明文（`vote_intro`）は、客観的・中立的な事実を担保しつつも、ユーザーの思考を刺激する対立軸をシャープに描き出します。

*   **疑問符（？）と対比構造の活用**
    *   冒頭または結びに「Aか、それともBか？」という明確な問いを置き、ユーザーの脳内に瞬時に「私はどっちだろう？」という自問自答を発生させます。
*   **文字数の制限:** **50文字〜70文字程度**（最大100文字）
    *   モバイル画面でスクロールせずに投票ボタンまで見渡せるよう、背景説明は極限まで短縮します。

## 3. 実装・運用におけるチェックリスト（今後の新規テーマ用）

今後新しいテーマの投票質問・選択肢を追加する際、以下のチェックリストをすべて満たしているか確認します。

- [ ] **スマホ実機での文字数確認:** 選択肢はすべて20文字前後に収まっており、折り返しても美しく配置されているか？
- [ ] **システム用語の排除:** 「その他」「別論点」「ノイズ」など、管理側のための分類用語が選択肢に残っていないか？
- [ ] **一次感情の含有:** 選択肢を選んだユーザーが「自分の心の内を代弁してくれている」と感じられるか？
- [ ] **中立なジレンマの提示:** 背景説明（Intro）が特定の立場を悪者として描いておらず、双方の「言い分・正義」を等しく刺激できているか？
- [ ] **葛藤の肯定:** 「選べない」という選択肢に、情報不足や対立の激しさに対するユーザーの心理（様子見、うんざり感、両論納得）が反映されているか？
